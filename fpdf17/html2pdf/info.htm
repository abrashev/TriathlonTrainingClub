Description

This is an enhancement of the WriteHTML() method from tutorial 6. Supported tags are:

<b> <i> <u> (as well as <strong> and <em>)
<a href="...">
<p> <br> (<tr> and <blockquote> treated like <br>)
<img src="..." width="..." [height="..."]>
<font face="..." color="...">
